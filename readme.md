# BICO Safe Module

Gnosis safe module for claiming vested `BICO` tokens, and staking from a safe.

## Usage

1. copy `.env.example` to `.env` and fill in the values.

```
cp .env.example .env
```

2. setup project

```
make
```

3a. run tests

```
make test
```

3b. run tests against mainnet fork

```
make test_f
```
